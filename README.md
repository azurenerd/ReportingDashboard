# Executive Reporting Dashboard

A single-page Blazor Server application that renders a pixel-perfect 1920×1080 project status dashboard for screenshot capture into PowerPoint decks. Visualizes project milestones on an SVG timeline and displays a color-coded heatmap grid of work items by status and month.

## Quick Start

### Prerequisites

- [.NET 8.0 SDK](https://dotnet.microsoft.com/download/dotnet/8.0) (any 8.0.x patch version)
- Microsoft Edge (Chromium) for screenshot capture

### Run the Dashboard
