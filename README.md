# Reporting Dashboard

Single-page, screenshot-optimized executive reporting dashboard (Blazor Server, .NET 8, Static SSR, local only).

## Getting Started

1. Install the [.NET 8 SDK](https://dotnet.microsoft.com/download/dotnet/8.0).
2. `git clone` this repo and `cd` into it.
3. `dotnet run --project src/ReportingDashboard.Web`
4. Open <http://localhost:5080/> in Edge or Chrome at 1920x1080.
5. Edit `src/ReportingDashboard.Web/wwwroot/data.json` and refresh the browser to see changes.
