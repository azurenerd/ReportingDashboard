# Executive Dashboard

A lightweight, screenshot-ready executive dashboard built with .NET 8 Blazor Server. Displays project milestones, work item status (shipped/in-progress/carried-over), and progress metrics on a single page. Renders pixel-perfect screenshots for PowerPoint presentations without post-processing.

## Quick Start

### Prerequisites
- .NET 8.0 SDK or later
- Modern browser (Chrome, Edge, or Firefox latest versions)
- Local machine (no cloud services required)
