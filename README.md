# Executive Dashboard

A lightweight, single-page executive reporting dashboard built with C# .NET 8 Blazor Server. Displays project milestones, progress metrics, and task status via a clean, screenshot-ready interface.

## Quick Start

### Prerequisites
- .NET 8 SDK ([download](https://dotnet.microsoft.com/download/dotnet/8.0))
- Windows 10+ or Linux (Ubuntu 20.04+)
- Modern browser (Chrome, Edge, Safari)
