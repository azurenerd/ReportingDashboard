# Executive Reporting Dashboard

A single-page Blazor Server dashboard that visualizes project milestones, delivery status, and blockers in a format optimized for PowerPoint screenshot capture. Renders at exactly 1920x1080 with no scrollbars, reading all data from a local `data.json` file. Zero external dependencies, zero infrastructure — just `dotnet run`.

## Prerequisites

- [.NET 8 SDK](https://dotnet.microsoft.com/download/dotnet/8.0) (8.0.x latest patch)
- A Chromium-based browser (Chrome, Edge)
- Windows 10/11 (primary target; macOS/Linux compatible but untested)
