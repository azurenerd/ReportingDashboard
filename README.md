# Executive Reporting Dashboard

A single-page Blazor Server (.NET 8) dashboard that visualizes project milestones on a timeline and displays a monthly execution heatmap. Optimized for 1920x1080 screenshot capture into PowerPoint decks.

## Prerequisites

- [.NET 8 SDK](https://dotnet.microsoft.com/download/dotnet/8.0) (8.0.x)
- Windows 10/11 (for Segoe UI font)
- Microsoft Edge or Google Chrome
