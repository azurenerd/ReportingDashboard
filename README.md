# Executive Project Reporting Dashboard

A lightweight, single-page executive reporting dashboard built with C# .NET 8 and Blazor Server. Visualizes project milestones, work item status, and health metrics from a JSON configuration file. Optimized for PowerPoint screenshots and local deployment.

## Features

- **Zero Authentication**: Local, single-user dashboard with no login required
- **Screenshot-Ready**: Professional styling optimized for PowerPoint and presentations
- **JSON-Driven**: All data sourced from simple JSON configuration file
- **Performance**: Sub-1-second page load times with in-memory caching
- **Self-Contained**: Deploy as single .exe executable with no external dependencies
- **Desktop-Optimized**: Designed for 1024x768 minimum, optimized for 1920x1080

## Quick Start

### Prerequisites
- .NET 8 SDK installed
- Chrome or Edge browser (latest 2 versions)
